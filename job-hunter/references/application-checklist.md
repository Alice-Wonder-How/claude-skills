# Pre-Send Checklist

Before handing an application back to Alice, verify:

- [ ] Every factual claim about the company is web-verified (not memory-based)
- [ ] No "built on top of X" or architecture claims unless confirmed
- [ ] People's names spelled correctly (ask Alice if unsure)
- [ ] Correct CV picked and stated
- [ ] Cover letter has zero em dashes
- [ ] No banned phrases (leveraging, passionate, excited to, look forward to)
- [ ] Opens with a fact, not a feeling
- [ ] Word/character limits respected for every form field
- [ ] AI systems builder + keynote speaker mentioned where natural
- [ ] LinkedIn contacts found: hiring manager + recruiter minimum
- [ ] Output is copy-pasteable block format, not a narrative about the draft
- [ ] Offered to iterate (shorter / more casual / more aggressive)
